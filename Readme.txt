Игра "Miner" - аналог игры "Сапер" для Windows.

Алгоритм работы программы:
Есть минное поле размером NxN.
Есть количество мин
Карта мин генерируется в случайном порядке.

При первом клике пользователь не должен подорваться. 

Минное поле строим случайным расположением мин.
1) Если это поле — мина, то подрываемся.
2) Если нет, то считаем сколько мин вокруг.
3) Если вокруг имеются мины, нужно отобразить их количество
Если вокруг нет мин, то это поле считается пустым и надо открыть все поля, которые находятся вокруг.
