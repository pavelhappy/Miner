Игра "Miner" - аналог игры "Сапер" для Windows.

Алгоритм работы программы(логическая часть):
Есть минное поле размером NxN.
Есть количество мин mineCount
Карта мин генерируется в случайном порядке.
Ячейка, граничащая с миной, nearMine, существует в восьми случаях
1) Мина слева от ячейки
2) Мина сверху-слева от ячейки
3) Мина сверху от ячейки
4) Мина сверху-справа от ячейки
5) Мина справа от ячейки
6) Мина снизу-справа от ячейки
7) Мина снизу от ячейки
8) Мина снизу-слева от ячейки

nearMine может быть одновременно рядом с несколькими минами, для каждой ячейки рассчитывается количество nearMine путем перебора всех граничащих ячеек.
При первом клике пользователь не должен подорваться. 

Минное поле строим случайным расположением мин.
1) Если это поле — мина, то подрываемся.
2) Если нет(nearMine = 0), то считаем сколько мин вокруг.
3) Если вокруг имеются мины, нужно отобразить их количество
Если вокруг нет мин, то это поле считается пустым и надо открыть все поля, которые находятся вокруг (рекурсивно проходим по пунктам 1-4)

Отдельно класс, реализующий логику.
Отдельно класс, рисующий поле.
